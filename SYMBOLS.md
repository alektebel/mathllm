## Pattern Recognition

MathDraw also intelligently recognizes these patterns:

### Fractions
- Input: `1/2` → Output: `\frac{1}{2}`
- Input: `x/y` → Output: `\frac{x}{y}`

### Exponents
- Input: `x^2` → Output: `x^{2}`
- Input: `a^n` → Output: `a^{n}`

### Subscripts
- Input: `x_1` → Output: `x_{1}`
- Input: `a_n` → Output: `a_{n}`

### Roots
- Input: `√2` → Output: `\sqrt{2}`
- Input: `√x` → Output: `\sqrt{x}`

### Absolute Value
- Input: `|x|` → Output: `\left|x\right|`

### Auto-sizing Delimiters
- Parentheses: `(x+y)` → `\left(x+y\right)`
- Brackets: `[a,b]` → `\left[a,b\right]`

## Total Support
- **150+ mathematical symbols**
- **All Greek letters** (uppercase and lowercase)
- **All common operators** (arithmetic, logic, set theory)
- **Calculus symbols** (integrals, derivatives, limits)
- **Smart pattern matching** (fractions, exponents, roots, etc.)

No matter what mathematical symbol you write, MathDraw will recognize it!
